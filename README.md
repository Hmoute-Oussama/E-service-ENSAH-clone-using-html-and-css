# ENSAH Website Clone

A premium, responsive website clone for the **École Nationale des Sciences Appliquées d'Al Hoceima (ENSAH)**.

##  Features
- **Modern Design**: Built with semantic HTML5 and clean CSS3.
- **Responsive Layout**: Optimized for desktop, tablet, and mobile devices.
- **Interactive Elements**: Includes a multimedia section with video/audio, interactive Google Maps, and a functional registration form.
- **Rich Aesthetics**: Uses the 'Inter' font family, smooth transitions, and a professional color palette.

##  Project Structure
- `index.html`: Homepage with school presentation and news.
- `formation.html`: Multimedia section showcasing school life.
- `inscription.html`: Detailed registration form for students.
- `contact.html`: Contact information and campus location.
- `assets/`: Contains all images, videos, and audio files.
- `style.css`: Global styles and design tokens.

##  Getting Started
1. Clone the repository.
2. Open `index.html` in any modern web browser.

##  Technologies Used
- HTML5
- CSS3 (Vanilla)
- Font Awesome (Icons)
- Google Fonts (Inter)
